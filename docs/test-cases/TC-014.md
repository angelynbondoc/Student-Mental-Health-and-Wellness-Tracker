# Test Cases: TC-014

**Project:** Student Mental Health & Wellness Tracker

| **Field**    | **Description**    |
|---|---|
| **Test ID**    | TC-014    |
| **Feature**    | Notification    |
| **Scenario**    | A user checks their notifications using the bell icon and views all notifications    |
| **Steps**    | 1. Sign in.    |
|    | 2. Look for the bell icon at the top right of the screen.    |
|    | 3. Click the bell icon to open the notification dropdown.    |
|    | 4. Check the list of recent notifications (e.g., liked, commented, shared).    |
|    | 5. Click "View all notifications" at the bottom of the dropdown.    |
|    | 6. On the Notifications page, use the filter tabs: All, Unread, Likes, Comments, Community, Announcements.    |
|    | 7. Click on a notification to go to the related post.    |
| **Expected Result**    | The bell icon shows a dropdown with recent notifications. Clicking "View all notifications" opens the full Notifications page with filter tabs. Clicking a notification should bring the user directly to the post it is about. |
| **Actual Result**    | The bell icon and notification dropdown are working. The full Notifications page with filter tabs (All, Unread, Likes, Comments, Community, Announcements) opens correctly. Clicking a notification brings the user directly to the related post. |
| **Status**    | PASS    |
| **Notes**    | Notifications are working correctly. Clicking a notification goes to the related post. |
