# Test Cases: TC-004

**Project:** Student Mental Health & Wellness Tracker  

| **Field**    | **Description**    |
|---|---|
| **Test ID**    | TC-004    |
| **Feature**    | Search & Knowledge Retrieval    |
| **Scenario**    | A user searches for posts and specific communities using the search feature.    |
| **Steps**    | 1. Sign in.    |
|    | 2. Go to the search bar.    |
|    | 3. Type a keyword to search for a post or content.    |
|    | 4. Verify the displayed results.    |
|    | 5. Type the name of a specific community.    |
|    | 6. Verify if the community appears in the search results.    |
| **Expected Result**    | Results that match the keyword appear. The searched community appears correctly in the results.    |
| **Actual Result**    | Search is working and shows the correct results for posts/content. Searching for a community name shows the correct community in the results.    |
| **Status**    | PASS    |
| **Notes**    | Search functionality is working as expected for both posts and communities.    |
