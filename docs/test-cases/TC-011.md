# Test Cases: TC-011

**Project:** Student Mental Health & Wellness Tracker  

| **Field**    | **Description**    |
|---|---|
| **Test ID**    | TC-011    |
| **Feature**    | Resource Library    |
| **Scenario**    | A user browses and reads an academic article in the Resource Library    |
| **Steps**    | 1. Sign in.    |
|    | 2. Go to the Resources section.    |
|    | 3. Browse the list of articles organized by year.    |
|    | 4. Click on an article to expand it and read the description.    |
|    | 5. Click the "Read article" button to open the full article.    |
| **Expected Result**    | The Resource Library shows a list of academic articles and readings sorted by year. Each article has a title, a short description, and a "Read article" button. Clicking the button opens the full article. The library can be visited anytime. |
| **Actual Result**    | The Resource Library is working. It shows academic articles and readings for student wellbeing, organized by year. Each article shows a title, a description, and a "Read article" button. Users can expand any article and read it anytime. |
