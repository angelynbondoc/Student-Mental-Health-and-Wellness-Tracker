## Test Cases: TC-016 

**Project:** Student Mental Health & Wellness Tracker

| Field | Description |
|-------|-------------|
| **Test ID** | TC-016 |
| **Feature** | Admin Moderator View |
| **Scenario** | The admin can view all posts and comments made by users, including the identities behind anonymous posts and comments. |
| **Steps** | 1. Admin logs into admin dashboard<br>2. Admin navigates to Moderator View page<br>3. Admin views all user posts<br>4. Admin views all user comments<br>5. Admin can see identities behind anonymous posts and comments<br>6. Admin can filter/search posts and comments as needed |
| **Expected Result** | Admin can view all posts and comments and see identities behind anonymous posts and comments |
| **Actual Result** | Admin successfully accessed all user posts and comments and was able to view the actual user identities behind posts and comments marked as anonymous |
| **Status** | Pass |
| **Notes** | Admin visibility features working as expected |
