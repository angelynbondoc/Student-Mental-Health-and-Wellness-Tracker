# Test Cases: TC-009

**Project:** Student Mental Health & Wellness Tracker 

| **Field**    | **Description**    |
|---|---|
| **Test ID**    | TC-009    |
| **Feature**    | Post Reactions and Comments    |
| **Scenario**    | User interacts with posts through liking, commenting, and deleting comments.    |
| **Steps**    | 1. Sign in to the application.    |
|    | 2. Find a post created by another user.    |
|    | 3. Click the Like button on the post.    |
|    | 4. Verify the like count increases by 1.    |
|    | 5. Scroll to the comment section of the post.    |
|    | 6. Enter a comment in the "Write a comment..." box.    |
|    | 7. (Optional) Enable "Post as anonymous."    |
|    | 8. Click the Submit/Post button.    |
|    | 9. Verify the comment appears under the post with appropriate details (username or anonymous, time, like, reply, delete options).    |
|    | 10. Locate the newly added comment.    |
|    | 11. Delete your own comment.    |
|    | 12. Verify the comment is removed successfully and delete button functions correctly.    |
|    | 13. Navigate to a post created by the logged-in user.    |
|    | 14. Add a comment from another account (or use an existing comment).    |
|    | 15. As the post owner, locate that comment.    |
|    | 16. Delete the comment.    |
|    | 17. Verify the comment is removed and that comment controls (reply, like, delete) are visible and functional before deletion.    |
| **Expected Result**   | - Like button increments the like count correctly.<br>- User can successfully add a comment (with or without anonymous option).<br>- Comments display correct details (username/anonymous, time, like, reply, delete buttons).<br>- All related buttons (like, reply, delete) are visible and functioning correctly.    |
| **Actual Result**    | Like count updates correctly. Comments are added and displayed correctly, including anonymous posting.    |
| **Status**    | PASS    |
| **Notes**    | Like count updates correctly. Comments are added and displayed correctly, including anonymous posting.    |
