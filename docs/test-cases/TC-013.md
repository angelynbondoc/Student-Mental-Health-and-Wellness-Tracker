# Test Cases: TC-013

**Project:** Student Mental Health & Wellness Tracker

| **Field**    | **Description**    |
|---|---|
| **Test ID**    | TC-013    |
| **Feature**    | Creating Post    |
| **Scenario**    | A user creates a new post using the Create tab    |
| **Steps**    | 1. Open the app.    |
|    | 2. Click or tap the Create tab in the menu bar.    |
|    | 3. Choose between New Post or New Community.    |
|    | 4. Select New Post.    |
|    | 5. Pick a community from the "Posting To" dropdown.    |
|    | 6. Type a message in the post box (up to 500 characters).    |
|    | 7. Optionally toggle "Post anonymously" to hide your name.    |
|    | 8. Click "Post to Community".    |
| **Expected Result**    | The Create page opens with two tabs: New Post and New Community. <br> The New Post form has a community selector, a post box with a 500 character limit, a "Post anonymously" toggle, and a "Post to Community" button. The post is submitted and appears in the feed |
| **Actual Result**    | The Create tab is working. The New Post form shows correctly with the community selector, post box (500 character limit), "Post anonymously" toggle, and "Post to Community" button. Posts are submitted successfully and appear in the community feed. |
| **Status**    | PASS    |
