# Test Cases: TC-010

**Project:** Student Mental Health & Wellness Tracker 

| **Field**    | **Description**    |
|---|---|
| **Test ID**    | TC-010    |
| **Feature**    | Mood Journal    |
| **Scenario**    | A user logs their mood and writes about how they are feeling    |
| **Steps**    | 1. Open the Journal section.    |
|    | 2. Pick a mood level: Very Low, Low, Neutral, Good, or Great.    |
|    | 3. Add context about why they feel that way.    |
|    | 4. Write about their feelings.    |
|    | 5. Save the entry.    |
| **Expected Result**   | The mood entry is saved and appears in the Past Entries section. A weekly mood chart also updates to show the user's mood over time.    |
| **Actual Result**    | All mood options (Very Low, Low, Neutral, Good, Great) are working. The entry goes through three steps: Mood, Context, and Write. Past entries are saved and visible. The weekly mood chart updates correctly.    |
| **Status**    | PASS    |
