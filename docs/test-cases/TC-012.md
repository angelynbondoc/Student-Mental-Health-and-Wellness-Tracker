# Test Cases: TC-012

**Project:** Student Mental Health & Wellness Tracker

| **Field**    | **Description**    |
|---|---|
| **Test ID**    | TC-012    |
| **Feature**    | Habit Tracker    |
| **Scenario**    | A user completes a daily habit in the Habit Tracker    |
| **Steps**    | 1. Open the Habits section.    |
|    | 2. Check the available habits:    |
|    |    - Morning Meditation (10 min) — complete 3 breathing cycles to mark as done    |
|    |    - Drink 8 glasses of water — tap each glass to log it    |
|    |    - 30-min walk or exercise — use the built-in timer    |
|    | 3. Complete the habit steps.    |
|    | 4. Try to reset or undo it.    |
| **Expected Result**    | Each habit has its own way of tracking completion. After finishing the habit you can do it again. The user can reset or undo it if needed. |
| **Actual Result**    | Habit Tracker is working. It has three built-in habits: Morning Meditation with a breathing exercise tracker, Drink 8 glasses of water with a tap-to-log feature, and 30-min walk or exercise with a built-in countdown timer. Each habit can be completed and reset. |
| **Status**    | PASS    |
| **Notes**    | Habits are pre-set and do not need to be entered by the user. Each habit has a unique tracker built in. |
