# Test Cases: TC-003

**Project:** Student Mental Health & Wellness Tracker

| **Field**    | **Description**    |
|---|---|
| **Test ID**    | TC-003    |
| **Feature**    | User Profile    |
| **Scenario**    | Verify that a user can manage their profile and personal content, including editing profile details, deleting posts, deleting their own comments, and leaving a community.    |
| **Steps**    | 1. Sign in to the application.    |
|    | 2. Navigate to the profile page.    |
|    | 3. Edit profile details:    |
|    |    a. Update username    |
|    |    b. Add/edit bio    |
|    |    c. Choose a profile picture or icon    |
|    |    d. Save changes    |
|    | 4. Verify updated profile information is displayed correctly.    |
|    | 5. Locate one of the user’s posts and delete it.    |
|    | 6. Confirm the post is removed.    |
|    | 7. Find a post where the user has commented.    |
|    | 8. Attempt to delete the user’s own comment.    |
|    | 9. Navigate to a community the user has joined.    |
|    | 10. Leave the community.    |
|    | 11. Refresh the page.    |
|    | 12. Verify the community no longer appears for the user.    |
| **Expected Result**    | - Profile updates are saved and displayed correctly.<br>- The user can delete their own posts successfully.<br>- The user can delete their own comments, and a delete option is available for their comments.<br>- After leaving and refreshing, the community is no longer visible to the user.<br>- Community list updates correctly after leaving    |
| **Actual Result**    | - Editing username, bio, and profile picture/icon works correctly.<br>- Users can delete their own posts.<br>- Users can delete their own comments; delete button is visible and functional.<br>- After leaving a community and refreshing, it no longer appears; browsing state updates correctly.    |
| **Status**    | PASS    |
| **Notes**    | The delete button appears only on the user’s own comments. Community removal reflects correctly after page refresh.    |
