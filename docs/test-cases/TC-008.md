# Test Cases: TC-008

**Project:** Student Mental Health & Wellness Tracker 

| **Field**    | **Description**    |
|---|---|
| **Test ID**    | TC-008    |
| **Feature**   | Community Feed    |
| **Scenario**    | A user selects a community from the Browse by Community bar and also checks the home page feed.    |
| **Steps**    | 1. Sign in.    |
|    | 2. Observe the Browse by Community bar at the top.    |
|    | 3. Click on a community (e.g., General, Programming & Dev, Gaming & Esports, Engineering & Making, or Books & Learning).    |
|    | 4. Check what posts appear.    |
|    | 5. Navigate to the home page.    |
|    | 6. Check the All Posts section and total post count.    |
|    | 7. Verify the Browse by Community bar is visible on the home page.    |
| **Expected Result**   | Only posts from the selected community are shown after clicking a community tab. The home page displays posts from all communities under the All Posts section. The total number of posts is shown. The Browse by Community bar is visible at the top of the home page.    |
| **Actual Result**    | The home page displays all posts. Clicking a community tab shows only posts from that community. Browse by Community bar is visible with options (e.g., General, Programming & Dev, Gaming & Esports, Engineering & Making, Books & Learning). Functionality is working correctly.    |
| **Status**    | PASS    |
| **Notes**    | Communities are displayed as clickable tabs at the top of the feed, making navigation straightforward.    |
