## Test Cases: TC-015

**Project:** Student Mental Health & Wellness Tracker


| **Field** | **Description** |
|-------|-------------|
| **Test ID** | TC-015 |
| **Feature** | Admin Reported Post |
| **Scenario** | Users who post suicidal thoughts will automatically trigger an alert to the admin. These posts will also appear on the admin's report page, where the admin can review and remove them if necessary. |
| **Steps** | 1. User creates a post with suicidal thoughts content<br>2. System automatically detects concerning content<br>3. Admin navigates to report page<br>4. Admin reviews the flagged post<br>5. Admin can remove post if necessary |
| **Expected Result** | Admin receives automatic alert and can review/remove posts with suicidal content |
| **Actual Result** | Admin received alert notification within 30 seconds of post creation and was able to successfully review and remove the flagged post from the admin dashboard |
| **Status** | Pass |
| **Notes** | Automatic detection system working correctly |
