# Test Cases: TC-001

**Project:** Student Mental Health & Wellness Tracker  

| **Field**    | **Description**    |
|---|---|
| **Test ID**    | TC-001    |
| **Feature**    | Privacy Notice on Sign-Up    |
| **Scenario**    | A student signs in to the app using their Gmail. A first-time user opens the app for the first time    |
| **Steps**    | 1. Open the app.    |
|    | 2. Click the Sign in with Gmail button.    |
|    | 3. Choose or confirm the Gmail account.    |
|    | 4. If a first-time user, check what appears after signing in.    |
| **Expected Result**    | The student is signed in and taken to the home page. First-time users see an onboarding screen with terms and conditions before continuing. |
| **Actual Result**    | Sign in using Gmail works correctly. Onboarding screen with terms and conditions appears for first-time users only.    |
| **Status**    | PASS    |
| **Notes**    | No email/password entry and no separate sign-up. Users sign in directly using their Gmail account. Onboarding only shows up the first time a user signs in. |
