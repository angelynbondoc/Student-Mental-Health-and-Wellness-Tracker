# Test Cases: TC-006

**Project:** Student Mental Health & Wellness Tracker 

| Field    | Description    |
|---|---|
| Test ID    | TC-006    |
| Feature    | Content Moderation (Admin)    |
| Scenario    | Admin manages reported posts (view, delete, dismiss, and check reporter)    |
| Steps    | 1. Sign in as an admin.    |
|    | 2. Go to the admin page.    |
|    | 3. View the list of reported posts.    |
|    | 4. Select a reported post and verify its details.    |
|    | 5. Check who reported the post.    |
|    | 6. Dismiss the report and confirm the post remains visible.    |
|    | 7. Select another reported post.    |
|    | 8. Delete the reported post.    |
| Expected Result   | • All reported posts are visible to the admin.<br>• The admin can view details of each report, including the reporting user.<br>• The admin can dismiss a report without removing the post.<br>• The admin can delete a reported post successfully.    |
| Actual Result    | • Reported posts are visible.<br>• Reporter details are displayed.<br>• Reports can be dismissed correctly.<br>• Reported posts can be deleted successfully.    |
| Status    | PASS    |
