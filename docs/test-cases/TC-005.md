# Test Cases: TC-005

**Project:** Student Mental Health & Wellness Tracker 

| **Field**    | **Description**    |
|---|---|
| **Test ID**    | TC-005    |
| **Feature**    | Recent Activity Tracking    |
| **Scenario**    | A user checks their past moods and journal entries in the past entries section |
| **Steps**    | 1. Sign in.    |
|    | 2. Go to the Journal section.    |
|    | 3. Scroll down to the Past Entries section.    |
|    | 4. Check if past moods and written content are visible.    |
| **Expected Result**   | The user can see a list of their past entries showing the mood they picked and what they wrote, along with a chart showing their mood this week.    |
| **Actual Result**    | Recent activity is working. Past entries are saved in the journal. The user can see the mood they picked (e.g., Very Low, Low, Neutral, Good, Great), what they wrote, and a chart showing their mood across the week.    |
| **Status**    | PASS    |
| **Notes**    | Activity tracking is saved in the journal only. Users can view past mood entries and a weekly mood chart.    |
